This file briefly explains the contents of the different directories
available at this level:

dll: Modules with focus on data link layer (e.g. WinProSt, URIS,
     WiMAC, PRIME)

loadgen: Load generator (the layer for the traffic being generated
         ranges from three to seven)

mock: Modules that serve mainly for testing

nl: Modules with focus on network layer (e.g. eHIS)

phy: Modules with focus on physical layer (e.g. OFDMAPhy, H2Phy)

tl: Modules with focus on transport layer (e.g. TCPIP)
